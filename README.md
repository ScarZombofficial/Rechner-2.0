Rechner 2.0 ist mein erstes kleines Projekt. Ich wollte es eigentlich immer weiter verbessern, aber es wurde lowkey langweilig. Weshalb ich nicht weiter daran arbeite. Ich habe sogar eine Website extra nur für das programmiert. Also viel Spaß damit.


README - Benutzen des Programmes: Rechner_2

-------------------------------------------------------------------------------------------------------
--Damit das Programm funktioniert musst du folgende Commands in den Cmd (Mit Admin rechten am besten) eingeben:

pip install --upgrade pip --user

pip install --user -U pygame

pip install --user -U pydub

winget install --id Gyan.FFmpeg -e --source winget --accept-package-agreements --accept-source-agreements

--Falls du auf Fehlermeldungen stößt, schau ob alles richtig ist und probiere es nochmal.
--Wenn ja und es weiterhin besteht, ignoriere es fürs erste und kontaktiere den Support.
-------------------------------------------------------------------------------------------------------

Wenn du auf Fehlermeldungen im Terminal stößt versuche zuerst das Programm neu zu installieren.
Falls das nicht klappt bitte kontaktiere den Support oder Developer.

Falls es ein Update gibt, einfach den alten "Rechner_2" Ordner mit dem neuen austauschen.
